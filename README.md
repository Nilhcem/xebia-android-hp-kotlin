Xebia Technical Job Interview Project entirely written in Kotlin

![Screenshot][1]

## Instructions

See `misc/instructions.pdf` (french only)

## Run Mock Server

```bash
cd mockserver
npm install
node server
```

You can then access it through:

```
http://localhost:8990/
```

## Run Unit Tests

```bash
./script/run-unit-tests.sh
```
